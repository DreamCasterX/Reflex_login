# Reflex_login
#### A simple login Web UI page created by Reflex 
- `python -m venv .venv` (optional)
- `source .venv/bin/activate` (optional)
- `pip install -r requirements.txt`
- `reflex run`
- Open your browser and visit `localhost:3000`<br/>
